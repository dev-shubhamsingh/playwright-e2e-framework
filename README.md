# Bliss

Bliss is an event management platform concept for discovering and publishing live experiences such as concerts, artist shows, parties, and large festival-style events.

## Current Features

- Event-focused homepage with clear product identity
- Event discovery search form (city + genre/vibe)
- Host flow to publish events (name, date, venue)
- Dashboard list that shows newly published events
- Responsive UI for desktop and mobile

## Tech Stack

- TypeScript (Node.js server)
- HTML, CSS, JavaScript (frontend)
- GitHub for version control and collaboration

## Run Locally

1. Clone the repository:
   ```bash
   git clone git@github.com:dev-shubhamsingh/playwright-e2e-framework.git
   ```

2. Open the project:
   ```bash
   cd playwright-e2e-framework
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open in browser:
   ```text
   http://127.0.0.1:3000
   ```

## Scripts

- `npm run dev` : Run TypeScript server directly
- `npm run build` : Compile TypeScript to `dist/`
- `npm start` : Run compiled server from `dist/`
- `npm run typecheck` : Validate TypeScript types
